Unique HTML Tags

<optgroup> — grouped dropdowns without JavaScript

optgroup wraps options under a label inside a dropdown. The label can't be selected — it's just a heading. Way more useful than it sounds for long lists.

<annotation-xml> — invisible note inside a math formula

annotation-xml attaches a hidden description to a formula — users don't see it but screen readers can. Kind of like alt text but for equations.

x = 5 x equals 5

<mtable> — basically a table but inside math

mtable works just like a regular HTML table — mtr is the row, mtd is the cell. Same thing, just with an m in front. Needed when you want to display a matrix inside MathML.

a b c d

<munderover> — puts things above and below a symbol

munderover takes 3 children — the symbol, what goes below, then what goes above. The order tripped me up at first. Great for summation signs where you need limits on the sigma.

∑ i=1 n ai
